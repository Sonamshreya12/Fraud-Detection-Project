# Fraud-Detection-Project
# Fraud Detection & Transaction Analysis Dashboard

## Project Overview
This project focuses on analyzing financial transaction data to identify fraud patterns and generate meaningful business insights using SQL and Power BI.

The dashboard provides interactive visualizations and detailed analysis of fraudulent transactions, transaction trends, risk categories, banks, states, and customer behavior.

---

## Objectives
- Detect fraudulent transaction patterns
- Analyze fraud trends across states and banks
- Identify high-risk transactions
- Create interactive dashboards for better decision-making
- Perform data analysis using SQL queries and Power BI visualizations

---

## Tools & Technologies Used
- Power BI
- PostgreSQL / SQL
- DAX
- Data Cleaning
- Data Visualization

---

## Dataset Features
The dataset contains:
- Transaction ID
- Customer ID
- Merchant Name
- Transaction Amount
- Fraud Risk
- Fraud Type
- State
- Card Type
- Bank
- Fraud Score
- Transaction Category

### Key SQL Operations:

- Card Type Analysis
- Fraud Risk vs Actual Fraud
- Monthly Fraud Trend
- High Risk Transactions
